== Description ==

The easiest way to integrate Exocep, the CRM service dedicated to winemakers, features to your WordPress site.
For now, you can use exomap shortcode to display a map of your resellers.
You need an account on http://www.exocep.com in order to obtain an API key and access these features.

= Translations =
* English - default
* French

= Demos =
* [ExoMap with Markers](https://exocep.fr/exomap/)

== Installation ==

Install the plugin and fill in the form in the setting page with the api key which was provided to you.
